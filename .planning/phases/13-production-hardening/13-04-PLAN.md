  1. Extract progress emission into a clean helper method:
     ```python
     async def _emit_progress(self, phase: str, step: str, pct: int, detail: str, **kwargs):
         """Unified progress emission — IPC queue + ProgressEmitter."""
         self._emit(phase, step, pct, detail, **kwargs)
         if self.use_progress_streaming and self._progress_emitter:
             if step in ("navigating", "scanning", "analyzing", "investigating", "deliberating"):
                 await self._progress_emitter.emit_agent_status(phase.title(), "running", detail)
             elif step == "done":
                 await self._progress_emitter.emit_agent_status(phase.title(), "complete", detail)
     ```

  2. Simplify the main loop by removing duplicate progress emission calls. Currently the loop has both `self._emit(...)` and `self._progress_emitter.emit_agent_status(...)` sprinkled everywhere. Use the unified `_emit_progress` instead.

     try:
         scout_update = await scout_node(state)

         state["errors"].append(f"Scout: {e}")
