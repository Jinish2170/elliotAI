# Plan 08-04: CTI-lite - IOCs Detection & MITRE ATT&CK

**Goal:** Implement cyber threat intelligence features for detecting Indicators of Compromise (IOCs), mapping to MITRE ATT&CK techniques, and providing threat attribution suggestions.

## Requirements

- CTI-03: Cyber Threat Intelligence mini-version (IOCs detection, threat attribution, MITRE ATT&CK)

## Context

Phase 8 requires CTI-lite functionality as a mini-version of full CTI integration. This provides threat detection capabilities through pattern recognition, IOC extraction, and alignment with the MITRE ATT&CK framework for standardized threat classification.

**Current State:**
- No IOC detection
- No MITRE ATT&CK integration
- No threat attribution
- Basic OSINT but no CTI intelligence

**Target State:**
- IOC detector extracts URLs, IPs, domains, email addresses, file hashes
- MITRE ATT&CK technique mapping for detected patterns

- CThreatIntelligence class integrates all CTI-lite features

Create CThreatIntelligence in veritas/osint/cti.py

Create CThreatIntelligence class:

1. __init__():
   - Initialize self.ioc_detector = IOCDetector()
   - Initialize self.attack_mapper = AttackPatternMapper()

2. async analyze_threats(url, page_html, page_text, page_metadata, osint_results) -> Dict:
   - Extract IOCs from text, html, metadata
   - Combine and deduplicate indicators
   - Classify threat level for each indicator
   - Extract site features from metadata
   - Map indicators to MITRE ATT&CK techniques
   - Generate attribution suggestion
   - Calculate overall threat level and confidence
   - Return dict with: target_url, indicators, mitre_techniques, attribution, threat_level, confidence

3. _extract_site_features(page_metadata) -> Dict:
   - Extract dark_patterns from metadata
   - Identify urgency patterns
   - Check for credential harvesting forms

4. _calculate_overall_threat(cti_result) -> tuple[str, float]:
   - Factor in MITRE techniques (confidence and count)
   - Factor in IOC threat levels (high/high threat IOC ratio)
   - Factor in attribution confidence
   - Weighted calculation: techniques 40%, threat_iocs 35%, attribution 25%
   - Return (threat_level, confidence_score)

Threat levels: critical (0.7+), high (0.5-0.7), medium (0.3-0.5), low (0.1-0.3), none (<0.1)
